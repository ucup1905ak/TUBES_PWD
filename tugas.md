## Tujuan Project

Projek kelompok ini bertujuan untuk menguji kemampuan mahasiswa dalam membangun sebuah aplikasi web yang dapat menyelesaikan permasalahan nyata.

---

## Ketentuan Project

1. Tugas dikerjakan secara **berkelompok** (5 mahasiswa per kelompok).
2. **Topik bebas**, namun **tidak boleh sama** antar kelompok.

---

## Deskripsi Projek

Buat aplikasi web yang memenuhi kriteria berikut:

- **Backend:** PHP murni (tanpa framework)
- **Frontend:** HTML, CSS, dan JavaScript
- **Fitur Utama:**
    1. Registrasi pengguna baru
    2. Enkripsi password saat registrasi
    3. Login ke sistem
    4. Melihat & mengupdate profil pengguna
    5. CRUD data melalui backend (minimal 2 operasi CRUD)

---

## Bonus Fitur

1. **Admin UI:** Tampilkan data backend melalui antarmuka admin
2. **Geolocation:** Tambahkan fitur geolokasi (relevan dengan permasalahan)
3. **Validasi Username/Email:** Deteksi jika username/email sudah dipakai saat registrasi
4. **Photo Profile:** Tambah/update foto profil pengguna
